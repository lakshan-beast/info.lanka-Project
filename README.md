# 🏛️ Info.Lanka – Sri Lanka Public Services Portal

Info.Lanka is a centralized, fully responsive web portal designed to provide quick and easy access to 200+ verified government and essential public service links in Sri Lanka. It simplifies the complexity of finding official government resources by organizing them into 10 intuitive categories.

## 🎯 Project Overview

In Sri Lanka, finding official government websites can be challenging due to outdated links or unofficial search results. Info.Lanka solves this by offering a curated, keyword-optimized search engine that connects citizens directly to verified portals for everything from Passports to Examination results.

## ✨ Key Features

- 🔍 Smart Search Engine: A high-performance filtering system that maps user intent using keywords (e.g., searching "water" finds "NWSDB").
- 🗂️ Categorized Navigation: Services organized into 10 key sectors (Health, Law, Education, Finance, etc.).
- ⚡ Quick Access (Top Services): One-click shortcuts for the most frequently used services like DMT, CEB, and Passport office.
- 📱 Fully Responsive: Optimized for Mobile, Tablet, and Desktop using CSS Grid and Flexbox.
- 🛡️ Security Best Practices: Implemented rel="noopener noreferrer" for all external links to ensure user safety.
- ⚙️ Modular Architecture: Built using ES6 Modules (Export/Import) for clean, maintainable code.

## 🛠️ Tech Stack

- HTML5: Semantic structure for better SEO and accessibility.
- CSS3: Custom properties (Variables), Grid, and Flexbox for modern layouts.
- JavaScript (ES6+): Modular programming, dynamic DOM manipulation, and advanced array filtering.
- Icons: Font Awesome for a modern visual hierarchy.

## 📂 Project Structure

├── index.html # Main portal structure
├── style.css # Custom styles & responsive design
├── script.js # Core logic & search functionality
├── servicesData.js # Centralized database of 200+ service links
└── README.md # Project documentation

## 🧠 Challenges & Learnings

- Data Management: Successfully managed a large dataset of 200+ objects by separating data from logic using ES6 Modules.
- Search Optimization: Developed a robust search algorithm that prioritizes user keywords over strict titles, significantly improving the UX.
- Professional UI/UX: Learned to implement industry-standard features like sticky headers, hover effects, and secure external linking.

## 🚀 How to Run Locally

1. Clone this repository.
2. Open the project folder in VS Code.
3. Use the Live Server extension to open index.html (Required for ES6 Module support).
